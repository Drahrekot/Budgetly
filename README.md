# ?? Budgetly

A clean, minimal budget tracker built with vanilla HTML, CSS & JavaScript.

## Features

- Add income and expense transactions with categories
- Live balance, total income, and total expense summary
- Spending breakdown with an animated doughnut chart (Chart.js)
- Persistent data with localStorage — survives page refresh
- Fully responsive on mobile and desktop

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- [Chart.js](https://www.chartjs.org/)
